Welcome to metrify’s documentation!¶

Overview¶

metrify é uma aplicação desenvolvida para scrum masters e equipes ágeis com o propósito automatizar a coleta e processamento de métricas relacionadas ao progresso de sprints e o cálculo e estimativa de rendimento com base em análise estatística do histórico de dados através do consumo da API do Github.

Nota

Este projeto está em fase de desenvolvimento ativo.
