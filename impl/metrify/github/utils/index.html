metrify.github.utils.generate_github_jwt(client_id: str, pem_path: str, expiration_time: int = 600) → str¶

Generates a JSON Web Token (JWT) for authenticating as a GitHub App from client ID and PEM file.

Parâmetros:

  • client_id (str) – Client ID for the Github App

  • pem_path (str) – Path to the PEM file containing a private key for the Github App.

  • expiration_time (int) – Expiration time for the JWT, in seconds; 600 second maximum limit.

Retorna:

JWT to authenticate the Github App

Tipo de retorno:

str

Levanta:

  • FileNotFoundError – if path specified for PEM file is invalid

  • UnsupportedKeyTypeError – if private key is invalid

  • InvalidArgumentError – if expiration_time argument is grater than 600 seconds
