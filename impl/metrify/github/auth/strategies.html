strategies – Funções de implementação do módulo¶

metrify/github/auth/strategies.py

The “strategies.py” module implements the “backend” for the scheduled jobs registered by the package’s “jobs.py”.

metrify.github.auth.strategies.get_access_token(jwt: str, installation_id: str) → str¶

Retrieves an installation access token from the Github API via an installation id and JWT.

Parâmetros:

  • jwt (str) – JWT generated from the Github App credentials

  • installation_id (str) – Github App installation ID

Retorna:

Github App installation access token

Tipo de retorno:

str
