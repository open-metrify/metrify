auth – Módulo de autenticação com a API¶

metrify/github/auth/__init__.py

The auth package implements the scheduling and backend routine functions to keep the application properly connected and authenticated to the Github API in order to execute queries via the global GraphQL client.

Sub-modules:

  • jobs – Rotinas executadas periodicamente
    • authenticate()
  • strategies – Funções de implementação do módulo
    • get_access_token()
